# My Book Notes Tracker

A personal website to keep track of books I’ve read, inspired by Derek Sivers.

## Features
- Add/edit/delete book entries
- Store notes, ratings, and reading date
- Fetch book covers using Open Library API
- Sort books by rating, title, or recency
- Responsive UI with dark mode

## Setup
1. Clone this repo
2. Run: `npm install`
3. Create a PostgreSQL DB and run `db/schema.sql`
4. Add your DB credentials to `.env`
5. Run: `nodemon index.js`# Book-Catalog-Manager
